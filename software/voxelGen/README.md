Compilation and Installation instructions:
=========================================

Debug mode:
------------------
```
make debug
```

Optimized mode:
----------------------
```
make optimized
```

Clean:
------
```
make clean
```

Execution:
----------
```
./voxelGen <with_split_mesh: 0 | 1> <num_threads> <input_file_absolute_path> <output_file_absolute_path>

```

These files are prepared as an 'exemplary' implementation of the C# codes in the repository. They are not optimized. 

To open the files you need to have a Rhino3D license. Grasshopper is free. 

Having a Rhino 5 and Grasshopper 0.9.0076 you can open these files as follows:

1. Open the .3dm file in Rhino
2. In Rhino, type in the command line "Grasshopper"
3. In Grasshopper, open the file with the .gh suffix. 
4. There are VB.NET and C# implentations of the algorithms inside this GH file. Plus several utility functions also written in C# & VB components. 
5. To see the source code of components (only the onesm which are programmed) double click on them. 
6. You can change the code and connect a panel to their outputs for debugging messages.
